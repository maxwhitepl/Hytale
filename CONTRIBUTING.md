# Wytyczne dotyczące współpracy z Hytale

Dziękujemy za współpracę z Hytale.
Podczas współpracy należy przestrzegać naszych **wytycznych dotyczących serwisu GitHub**.

### Dlaczego ta informacja się tu znajduje?
Umieściliśmy ją tutaj, aby serwis GitHub powiadamiał Cię podczas przesyłania pull requestu o zmianach w wytycznych dotyczących PR, jeśli nie zapoznałeś się z najnowszą wersją.
